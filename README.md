# Employee Management System

A simple Java console-based Employee Management System developed using Object-Oriented Programming (OOP).

## Features

- Add a new employee
- View all employees
- Update employee information
- Delete an employee
- Validate user input
- Prevent duplicate employee IDs

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Scanner
- Git and GitHub
